Endpoint for gp_ei POST requests.

POST /gp/ei¶

Calculates the Expected Improvement (EI) of a set of points, given historical data.

JSON Parameters:
 
  • points_to_evaluate (list) – points to calculate EI for
Status Codes:
  • 200 – returns a response
  • 500 – server error
pretty_default_request = {'points_to_evaluate': [[0.1], [0.5], [0.9]], 'gp_info': {'points_sampled': [{'value_var': 0.01, 'value': 0.1, 'point': [0.0]}, {'value_var': 0.01, 'value': 0.2, 'point': [1.0]}], 'domain': [[0, 1]]}}¶
