Module contents¶

Testing code for the (Python) bandit library.

Testing is done via the Testify package: https://github.com/Yelp/Testify

This package includes:

  • Test cases/test setup files
  • Tests for bandit/epsilon
  • Tests for bandit/ucb

Files in this package

  • bandit_test_case.py: test case with different sampled arm historical info inputs

This package includes:

  • Test cases/test setup files
  • Tests for classes and utils in moe.bandit

Files in this package

  • moe.tests.bandit.bandit_interface_test: tests for moe.bandit.interfaces.bandit_interface.BanditInterface
  • moe.tests.bandit.bandit_test_case: base test case for bandit tests with a simple integration test case
  • moe.tests.bandit.epsilon_first_test: tests for moe.bandit.epsilon_greedy.EpsilonFirst
  • moe.tests.bandit.epsilon_greedy_test: tests for moe.bandit.epsilon_greedy.EpsilonGreedy
  • moe.tests.bandit.epsilon_test: tests for moe.bandit.epsilon_greedy.Epsilon
  • moe.tests.bandit.epsilon_test_case: test cases for classes under moe.bandit.epsilon.Epsilon
  • moe.tests.bandit.linkers_test: tests for moe.bandit.linkers
